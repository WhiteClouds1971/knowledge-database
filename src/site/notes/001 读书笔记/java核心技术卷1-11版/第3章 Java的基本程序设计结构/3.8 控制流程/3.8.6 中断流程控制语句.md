# break

break 用于退出 switch 和循环
## 带有标签的 break

Java 还提供了一种带标签的 break 语句，用于跳出多重嵌套的循环语句。

```java
break read_data:
while(...){
	for(...){
		break read_data;
	}
}
```

标签必须放在希望跳出的最外层循环之前，并且必须**紧跟一个冒号**。
