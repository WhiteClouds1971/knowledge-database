```java
switch (choice){
	case 1:break; 
	case 2:break;
	case 3:break;
	case 4:break;
	default:break;
}
```

switch 语句将从与选项值相匹配的 case 标签开始执行，直到遇到 break 语句，或者执行到 switch 语句的结束处为止。如果没有相匹配的 case 标签，而有 default 子句，就执行这个子句。

case 标签可以是:

- 数字 char、byte、short 或 int 的常量表达式。
- 枚举常量。
- 从 Java7 开始，case 标签还可以是字符串字面量。
