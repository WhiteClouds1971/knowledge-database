1. 使用 final 指示常量，例如：

```java
final double CM_PERINCH = 2.54;
```

关键字 final 表示这个变量只能被赋值一次。习惯上，常量名全大写，用_连接。

>[! note] const 是 Java 保留的关键字，但目前并没有使用。在 Java 中，必须使用 final 定义常量。
