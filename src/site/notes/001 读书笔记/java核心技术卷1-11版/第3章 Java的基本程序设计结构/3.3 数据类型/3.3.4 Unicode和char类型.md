**Unicode 需要解决的问题？**
{ #24adec}


- 一是在不同的编码方案下有可能对应不同的字母。如何 ASCII 和 GB 18030 中相同的编码可能代表不同的字符。
- 二是采用大字符集的语言其编码长度有可能不同。例如，有些常用的字符采用单字节编码，而另一些字符则需要两个或多个字节。
