>x+=4 等价于 x = x + 4

>[! tip] 如果运算符得到一个值，其类型与左侧操作数的类型不同，就会发生强制类型转换。例如，如果 x 是一个 int ，则以下语句 x += 3.5;  是合法的，将把 x 设置为 (int)(x + 3.5)。
