从概念上讲，Java 字符串就是 Unicode 字符序列。例如，字符串“Java\\u212"由 5 个 Uni code 字符 J、a、v、a 和 <sup>TM</sup> 组成。Java 没有内置的字符串类型，而是在标准 Java 类库中提供了一个预定义类，很自然地叫做 String。每个用双引号括起来的字符串都是 String 类的一个实例:

String greeting = "Hello";
