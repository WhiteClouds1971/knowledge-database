数组元素的下标为从 0 ~ 99 (不是 1 ~ 100 )。

```java
int[] a = new int [100];
for (int i=0; i<100; i++)
	a[i] = i; // fills hte array with numbers 0 ot 9
```

创建数组未初始化时的默认值：
{ #18d161}

- 数字：0
- boolean：false
- 对象：null
- 字符串：null，String 也是一种类
