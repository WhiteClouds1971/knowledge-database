# 逻辑运算符

`==`（等于）；!=（不等于）；<（小于）；>（大于）；<=（小于等于）；>=（大于等于）；&&（逻辑与）；||（逻辑或）；!（逻辑非）；?:（三元运算符）

其中的&&和||是按照短路的方式来求值：如果第一个操作数已经能够确定表达式的值，第二个操作数就不必计算了。
{ #474ec4}


可以利用短路来避免一些错误：

```java
x!=0&&1/x+y // no division by 0
```

如果 x 等于 0，那么第二部分就不会计算。因此，如果 x 为日，也就不会计算 1/x，除以 0 的错误就不会出现。
