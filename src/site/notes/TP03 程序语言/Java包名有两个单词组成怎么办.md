在 Java 中，当包名由两个单词组成时，我们可以使用下划线分隔两个单词或者使用首字母大写的驼峰命名法来命名包名。无论哪种方式，都需要保持包名的唯一性，并且符合 Java 的命名规范。

| 方式    | 示例             |
| ----- | -------------- |
| 下划线分割 | // com_example |
| 驼峰法   | // ComExample  |
